### 前端性能优化的目的

从用户角度而言，优化能够让页面加载得更快、对用户的操作响应得更及时，能够给用户提供更为友好的体验。 从服务商角度而言，优化能够减少页面请求数、或者减小请求所占带宽，能够节省可观的资源。

### 前端性能优化的方法

页面部分
<!--more-->
+ 减少Http请求数量
+ 减少DNS查询次数
+ 尽量避免ifreme框架的使用
+ 避免网站出现404

JavaScript

+ 将脚本置于底部
+ 使用外部JavaScript
+ 合并脚本
+ JavaScript压缩

CSS

+ 样式表置于页面顶部
+ 使用link代替@import
+ 避免使用表达式

图片

+ 压缩图像大小
+ 合并 CSS图片
+ 合理使用字体图标
+ 对图片进行懒加载处理

服务器端

+ 使用CDN
+ 开启Gzip压缩
+ 避免图片空连接
+ 减少跳转
+ 避免重复请求
